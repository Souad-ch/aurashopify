# Aura — منصة تجارة إلكترونية (Shopify Clone)

**Aura** منصة تجارة إلكترونية متكاملة شبيهة بـ Shopify، تتيح لأي تاجر بناء متجره
الإلكتروني وإدارة المنتجات والطلبات والعملاء، وتنمية أعماله — كل ذلك من لوحة تحكم
واحدة. مبنية بـ **Next.js 14** و **Prisma** و **Tailwind CSS**.

## ✨ المميزات

### 🏪 واجهة المتجر (Storefront)
- صفحة متجر عامة لكل تاجر على `/store/[slug]`
- عرض المنتجات وصفحة تفاصيل لكل منتج
- سلة تسوق كاملة (مع حفظ في `localStorage`)
- إتمام الشراء (Checkout) ينشئ طلباً وعميلاً حقيقيين في قاعدة البيانات
- ألوان وهوية المتجر قابلة للتخصيص

### 📊 لوحة تحكم التاجر (Dashboard)
- **نظرة عامة**: إحصائيات الإيرادات والطلبات والعملاء + رسم بياني للإيرادات
- **المنتجات**: إضافة/تعديل/حذف منتجات مع المخزون والتسعير
- **الطلبات**: عرض جميع الطلبات وحالاتها
- **العملاء**: قائمة العملاء وإجمالي إنفاقهم
- **التحليلات**: رسوم بيانية (شهرية، حسب التصنيف) وأفضل المنتجات
- **الفوترة والاشتراك**: إدارة الخطة وسجل الفواتير
- **الإعدادات**: تخصيص معلومات المتجر والعملة والألوان

### 💳 المصادقة والاشتراكات
- تسجيل دخول/إنشاء حساب (جلسات JWT في كوكيز httpOnly)
- ثلاث خطط اشتراك (Starter / Growth / Advanced)
- تبديل بين الفوترة الشهرية والسنوية
- اشتراك وإلغاء فعليان محفوظان في قاعدة البيانات

## 🛠️ التقنيات

| الطبقة | التقنية |
|--------|---------|
| الإطار | Next.js 14 (App Router, Server Actions) |
| اللغة | TypeScript |
| قاعدة البيانات | SQLite عبر Prisma ORM |
| التنسيق | Tailwind CSS |
| الرسوم البيانية | Recharts |
| المصادقة | JWT (jose) + bcryptjs |
| التحقق | Zod |

## 🚀 التشغيل محلياً

```bash
# 1. تثبيت الحزم
npm install

# 2. إعداد قاعدة البيانات وزرع البيانات التجريبية
npm run db:push
npm run db:seed

# 3. تشغيل بيئة التطوير
npm run dev
```

ثم افتح [http://localhost:3000](http://localhost:3000).

## 📁 بنية المشروع

```
src/
├── app/
│   ├── page.tsx                # الصفحة الرئيسية (تسويقية)
│   ├── pricing/                # صفحة الأسعار والاشتراك
│   ├── login/ , register/      # المصادقة
│   ├── dashboard/              # لوحة تحكم التاجر
│   │   ├── products/ , orders/ , customers/
│   │   ├── analytics/ , billing/ , settings/
│   ├── store/[slug]/           # واجهة المتجر العامة
│   └── actions/                # Server Actions (auth, products, store, checkout, subscription)
├── components/                 # مكونات الواجهة (dashboard, store, ...)
└── lib/                        # prisma, auth, format
prisma/
├── schema.prisma               # نماذج البيانات
└── seed.mjs                    # بيانات تجريبية
```
