# Aura — منصة تجارة إلكترونية (Shopify clone)

**Aura** منصة تجارة إلكترونية متكاملة شبيهة بـ Shopify: واجهة متجر للزبائن،
لوحة تحكم للتاجر، ولوحة إدارة للمنصة — مبنية بـ **Next.js 14** و **Tailwind CSS**.

## 🔗 الموقع المباشر (Live Demo)

بعد تفعيل GitHub Pages، يكون الموقع متاحاً على:

**https://souad-ch.github.io/aurashopify/**

> هذه نسخة عرض (Demo) ثابتة تعمل بالكامل في المتصفح ببيانات تجريبية
> (لا يوجد حفظ دائم). نسخة قاعدة البيانات الحقيقية موجودة على فرع
> `fullstack-version` ويمكن نشرها على Vercel.

## ✨ الأقسام

- 🏠 **موقع تعريفي**: صفحة رئيسية + صفحة أسعار وخطط اشتراك
- 🛍️ **متجر للزبائن** (`/store/aura-boutique`): منتجات، صفحة منتج، سلة تسوق، إتمام شراء
- 📊 **لوحة تحكم التاجر** (`/dashboard`): نظرة عامة بالرسوم البيانية، منتجات مع بحث/تصفية، طلبات، عملاء، تحليلات، فوترة، إعدادات
- 🛰️ **لوحة إدارة المنصة** (`/admin`): إحصائيات، متاجر، مستخدمون، خطط وإيرادات

## 🚀 التشغيل محلياً

```bash
npm install
npm run dev      # http://localhost:3000
```

لبناء النسخة الثابتة:

```bash
npm run build    # ينتج مجلد out/
```

## ⚙️ تفعيل GitHub Pages (مرة واحدة)

1. ادفع الكود إلى الفرع `main`.
2. من إعدادات المستودع → **Settings → Pages**.
3. اختر **Source: GitHub Actions**.
4. سيعمل التدفّق تلقائياً وينشر الموقع على الرابط أعلاه.
